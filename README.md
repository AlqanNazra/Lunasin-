Lunasin adalah aplikasi mobile manajemen pelunasan utang/piutang yang dirancang untuk membantu pengguna mengatur keuangan secara lebih terorganisir dan efisien. 
Aplikasi ini memungkinkan pengguna mencatat, memantau, dan menyelesaikan kewajiban finansial secara praktis dan transparan.

A. Fitur Utama
1. Pencatatan Utang dan Piutang: Tambah data utang/piutang dengan detail lengkap.
2. Pengingat Jatuh Tempo: Notifikasi otomatis sebelum utang jatuh tempo.
3. Ringkasan Keuangan: Lihat statistik utang/piutang aktif dan yang sudah lunas.
4. Manajemen Kontak: Tambahkan data pihak pemberi/penerima utang.
5. Keamanan Data: Data disimpan secara terenkripsi (jika ada).
6. Antarmuka Responsif: Kompatibel di desktop maupun mobile.

B. Teknologi yang digunakan 
Frontend: Kotlin framework jackcompose
Backend: Kotlin, FCM
Database: Firestore
Bahasa Pemrograman: Java & Kontlin.
